# 비-개발자 프로젝트 참여 방법

- issue 등록 방법은 [개발 프로세스 - 0. Issue 작성 (upstream)](https://github.com/knut-cs/Curriculum/wiki/개발-프로세스#0-issue-작성-upstream) 을 참조해주세요.
